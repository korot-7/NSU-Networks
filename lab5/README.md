# Лабораторная работа: SOCKS-прокси

## Описание
Цель лабораторной работы — Необходимо реализовать прокси-сервер, соответствующий стандарту SOCKS версии 5, с реализацией только команды 'establish a TCP/IP stream connection' , а также использование неблокирующихся сокетов, работая с ними в рамках одного треда.

В данной лабораторной для DNS резолвера мы используем Google DNS (адрес: '8.8.8.8', порт: '53'). Если данный резолвер не устраивает, то поменять эти данные вы можете в src/main/java/constants/Constants

---

## Запуск
- Если запускать в IDEA, то нужно только указать в **Run Configurations** в **Program argumetns**: `<port>`
  `<port> нужен порт, на котором прокси будет ждать входящих подключений от клиентов.`
