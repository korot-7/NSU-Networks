# Лабораторная работа: Места (Асинхронное сетевое взаимодействие)

## Описание
Цель лабораторной работы — Используя методы асинхронного программирования 
провзаимодействовать с несколькими публично доступными API и сделать приложение с интерфейсом, 
основанное на этих API.

---

## Требования
- Установлен JavaFX SDK, иначе скачать JavaFX SDK

---

## Запуск
- Если запускать в IDEA, то нужно только указать в **Run Configurations** в **VM options**: `<--module-path "C:\javafx-sdk-25\lib" --add-modules=javafx.controls>`
  Только указать путь до своей JavaFX SDK
